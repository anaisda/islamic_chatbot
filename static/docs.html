ISLAMIC LIBRARY · RAG API · v3.0.0

Two Scholars,
One API

Semantic search over the complete libraries of Sheikh Ibn Baz and Sheikh Ibn Othaymeen رحمهما الله. Ask in Arabic, get verbatim citations with exact book and page references.

Base URL https://islamic-chatbot-ukbz.onrender.com
Scholars Ibn Baz · Ibn Othaymeen
Model gpt-4o-mini

Authentication

All /v1/ endpoints require an API key in the request header.

HTTP Header
X-API-Key: your-secret-key
ℹ️ The web interface uses /api/chat which is public. Your app must use /v1/ endpoints with the API key set in Render → Environment → CHATBOT_API_KEY.

Scholar Filter

All /v1/ endpoints accept an optional scholar parameter to filter results.

null → both scholars ibn_baz → Ibn Baz only ibn_othaymeen → Ibn Othaymeen only
ValueDescription
null / omitSearch both scholars, show both opinions
ibn_bazFilter to Ibn Baz رحمه الله only
ibn_othaymeenFilter to Ibn Othaymeen رحمه الله only

Dart / Flutter

Dart · http package
import 'dart:convert';
import 'package:http/http.dart' as http;

const baseUrl = 'https://islamic-chatbot-ukbz.onrender.com';
const apiKey  = 'your-secret-key';

// Both scholars
Future<Map> askBoth(String question) async {
  final res = await http.post(
    Uri.parse('$baseUrl/v1/query'),
    headers: {'Content-Type': 'application/json', 'X-API-Key': apiKey},
    body: jsonEncode({'question': question}),
  );
  return jsonDecode(res.body);
}

// One scholar only
Future<Map> askScholar(String question, String scholar) async {
  // scholar: 'ibn_baz' or 'ibn_othaymeen'
  final res = await http.post(
    Uri.parse('$baseUrl/v1/query'),
    headers: {'Content-Type': 'application/json', 'X-API-Key': apiKey},
    body: jsonEncode({'question': question, 'scholar': scholar}),
  );
  return jsonDecode(res.body);
}

// Usage
final both = await askBoth('ما حكم الزكاة على الذهب؟');
final baz  = await askScholar('ما حكم الزكاة؟', 'ibn_baz');
final oth  = await askScholar('ما حكم الزكاة؟', 'ibn_othaymeen');

Python

Python · requests
import requests

BASE = "https://islamic-chatbot-ukbz.onrender.com"
H    = {"Content-Type": "application/json", "X-API-Key": "your-secret-key"}

# Both scholars
r = requests.post(f"{BASE}/v1/query", headers=H,
    json={"question": "ما حكم تارك الصلاة؟"})
print(r.json()["answer"])

# Ibn Baz only
r = requests.post(f"{BASE}/v1/query", headers=H,
    json={"question": "ما حكم تارك الصلاة؟", "scholar": "ibn_baz"})
print(r.json()["answer"])

# Ibn Othaymeen only
r = requests.post(f"{BASE}/v1/query", headers=H,
    json={"question": "ما حكم تارك الصلاة؟", "scholar": "ibn_othaymeen"})
print(r.json()["answer"])

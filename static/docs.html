ISLAMIC LIBRARY · RAG API

Ibn Baz Fatwa
Search API

Semantic search over the complete library of Sheikh Ibn Baz رحمه الله. Ask questions in Arabic and receive verbatim citations with exact book and page references.

Base URL https://islamic-chatbot-ukbz.onrender.com
Vectors 27,000+
Model gpt-4o-mini

Authentication

All /v1/ endpoints require an API key. Pass it in the request header.

ℹ️ The /api/chat and /api/status endpoints are public (used by the web interface). Your app should use /v1/ endpoints with the API key.
HTTP Header 
X-API-Key: your-secret-key
⚠️ Set your API key in Render → Environment → CHATBOT_API_KEY. Never expose it in client-side code.

Endpoints

POST /v1/query Ask a question, get an AI answer with citations 🔑 auth

Performs semantic search over the Ibn Baz library, then uses GPT-4o-mini to generate a response with verbatim citations. Quranic verses and hadiths are automatically corrected to their canonical form.

Request Body

ParameterTypeRequiredDescription
questionstringrequired The question in Arabic (max 1000 chars)
top_kintegeroptional Number of source passages to retrieve (default: 5, max: 10)
sources_onlybooleanoptional If true, skip LLM generation and return raw passages only (faster, cheaper)

Response

  "question":   "ما حكم تارك الصلاة؟",

  "sources": [
    {
      "book":    "مجموع فتاوى ابن باز - الجزء العاشر",
      "page":    "142",
      "scholar": "Ibn Baz",
      "text":    "النص الحرفي من الكتاب...",
      "score":   0.8821
    }
  ],
  "latency_ms": 1243,
  "model":      "gpt-4o-mini",
  "top_k":      5
}

Code Examples

Python

Python · requests 
import requests

BASE_URL = "https://islamic-chatbot-ukbz.onrender.com"
API_KEY  = "your-secret-key"

headers = {
    "Content-Type": "application/json",
    "X-API-Key":     API_KEY
}

# Ask a question
response = requests.post(
    f"{BASE_URL}/v1/query",
    headers=headers,
    json={
        "question": "ما حكم تارك الصلاة؟",
        "top_k":    5
    }
)

data = response.json()
print(data["answer"])

for source in data["sources"]:
    print(f"{source['book']} | ص {source['page']} | score: {source['score']}")

# Search only (no LLM)
r = requests.post(
    f"{BASE_URL}/v1/search",
    headers=headers,
    json={"query": "الزكاة", "top_k": 3}
)
print(r.json())

JavaScript / TypeScript

JavaScript · fetch 
const BASE_URL = "https://islamic-chatbot-ukbz.onrender.com";
const API_KEY  = "your-secret-key";

async function askQuestion(question) {
  const res = await fetch(`${BASE_URL}/v1/query`, {
    method:  "POST",
    headers: {
      "Content-Type": "application/json",
      "X-API-Key":     API_KEY
    },
    body: JSON.stringify({ question, top_k: 5 })
  });

  const data = await res.json();
  return data;
}

async function search(query) {
  const res = await fetch(`${BASE_URL}/v1/search`, {
    method:  "POST",
    headers: {
      "Content-Type": "application/json",
      "X-API-Key":     API_KEY
    },
    body: JSON.stringify({ query, top_k: 5 })
  });
  return res.json();
}

// Usage
const result = await askQuestion("ما حكم صيام المريض؟");
console.log(result.answer);
result.sources.forEach(s => console.log(s.book, s.page));

Dart / Flutter

Dart · http package 
import 'dart:convert';
import 'package:http/http.dart' as http;

const baseUrl = 'https://islamic-chatbot-ukbz.onrender.com';
const apiKey  = 'your-secret-key';

Future<Map<String, dynamic>> askQuestion(String question) async {
  final response = await http.post(
    Uri.parse('$baseUrl/v1/query'),
    headers: {
      'Content-Type': 'application/json',
      'X-API-Key':     apiKey,
    },
    body: jsonEncode({
      'question': question,
      'top_k':    5,
    }),
  );

  if (response.statusCode == 200) {
    return jsonDecode(response.body);
  } else {
    throw Exception('API error: ${response.statusCode}');
  }
}

// Usage in Flutter widget
final result = await askQuestion('ما حكم الزكاة على الذهب؟');
final answer  = result['answer'] as String;
final sources = result['sources'] as List;
